# jQuery 事件

# 绑定事件处理器

## on

描述: 在选定的元素上绑定一个或多个事件处理函数。（可用于事件委托）

比如：
```js
$(document).ready(function() {
  $('ul').on('click', 'li', function() {
    $(this).slideToggle();
  });
});
```
其中，`li` 就是委托者，只要在 ul 内的 li 都会被绑定。（通过事件冒泡）

## keyup事件

当用户在一个元素上释放按键(是键盘的键~不是鼠标键)的时候，keyup事件就会被附加到这个元素。

它可以绑定到任何元素，但该事件只是发送到获取焦点的元素上。不同的浏览器中，可获得焦点的元素略有不同，但是表单元素总是能获取焦点，所以对于此事件类型表单元素是最合适的。

## focus事件

当一个元素获得焦点时，focus事件被触发。获取焦点的元素，浏览器一般会以某种高亮方式来显示，比如，点线围绕的元素。该事件通常用于判断哪个元素是最先接收键盘相关事件的元素。
