# HTML 技巧

## HTML 注释例子：

```html
<body>

  <!--==========================
    Header
  ============================-->
  <header>
    <section><section>
    <section><section>
    <section><section>
    <section><section>
  </header><!-- #header -->


  <!--==========================
    main
  ============================-->
  <main id="main">
    <section><section>
    <section><section>
    <section><section>
  </main><!-- #main -->

  <!--==========================
    Footer
  ============================-->

  <footer id="footer">
    <section><section>
    <section><section>
    <section><section>
    <section><section>
  </footer><!-- #footer -->

</body><!-- body -->
```

- body 里面包含3部分，header、main、footer 这样就是能更好的语义化~
