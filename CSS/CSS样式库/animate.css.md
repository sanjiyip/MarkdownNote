# animate.css

## 一个跨浏览器的 css 动画库，简单易用

### 使用方法：

- 在 html 文档中引入 animate.css 文件

```html
<link rel="stylesheet" href="./animate.min.css">
```

- html 元素中加入 `class = "animate"` 启动动画，再添加相应的动画效果的 class 值，动画就能生效。比如：

```html
<h1 class="animated infinite bounce">Example</h1>
```

> 表示是元素 h1 一直弹跳的动画。
